Camel Mina Router Project
====================

To run this router either embed the jar inside Spring
or to run the route from within maven try

    mvn camel:run

The sample client code is available under test-classes.

You may try running the client using

java com.fusesource.fusebyexample.TCPClient

For more help see the Apache Camel documentation

    http://camel.apache.org/
